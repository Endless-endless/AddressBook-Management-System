# 通讯录管理系统

## 📌 项目简介

本项目是一个基于 C++ 开发的控制台通讯录管理系统，使用 Visual Studio 2022 编写，实现了对联系人信息的基本管理功能。

## 🚀 功能介绍

- 添加联系人
- 删除联系人
- 查找联系人
- 修改联系人
- 显示全部联系人
- 最多支持 1000 条联系人数据

## 🛠 技术栈

- C++
- 结构体（struct）
- 控制台菜单交互
- 静态数组存储

## 📂 项目结构

- `通讯录管理系统.cpp` ：核心程序代码
- `.vcxproj` ：Visual Studio 项目文件

## ▶ 运行方式

1. 使用 Visual Studio 2022 打开 `.sln` 或 `.vcxproj`
2. 编译运行（Ctrl + F5）

## 📖 设计思路

- 使用 `Person` 结构体保存联系人信息
- 使用 `Addressbooks` 结构体管理联系人数组
- 通过菜单循环实现功能选择

## 📌 项目特点

- 逻辑清晰
- 结构简单
- 适合 C++ 初学者练习数据结构与函数封装
